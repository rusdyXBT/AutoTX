## 📦 Install

```bash
git clone https://github.com/rusdyXBT/AutoTX
cd AutoTX
npm install
```

---

## ⚙️ Setup

Create `.env`:

```env
PRIVATE_KEY=your_private_key_here
```

---

## 🔹 One → Many

Edit `address.txt`:

```
0xAddress1
0xAddress2
0xAddress3
```

---

## 🔹 Many → One

Edit `pk.txt`:

```
private_key_1
private_key_2
private_key_3
```

---

## ▶️ Run

```bash
node main.js
```

---

## ⚠️ Notes

- Keep private key safe
- Don't upload sensitive files
- Test with small amount first
